# SPOmiAlign
spatial multimodal alignment including matching_warpping.py align_h5ad.py and reassignment.py. 
The input data of matching_warpping.py are two images and h5ad files if spatial omic section exists. The output data are aligned images and warpped csv files of the source section spots location. 

The input of align_h5ad.py is source csv files and output is aligned source section h5ad files.

The input of reassignment.py is reference and source h5ad files and the output are the two reassigned sections h5ad files.
